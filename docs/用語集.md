## ランク判定の閾値（コード/ドキュメント共通）— Quality-Gate Shima-Final（#96）

- **S ランク**: 思考密度 ≥ 70 + 稼働日数 ≥ 30 + 意思シグナル ≥ 3件 + 実稼働コード ✓ + テスト証跡 ✓
- **A ランク**: 思考密度 ≥ 60 かつ 稼働日数 ≥ 7（または S 条件の一部未達）
- **B ランク**: 上記未満（最低保証ランク）

## Trust Score の重み（v1）
- 品質履歴（過去登記の平均ランク重み）: 50%
- Discord 貢献度: 30%
- X 拡散実績: 20%
