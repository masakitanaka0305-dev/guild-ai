# Pro Search 設計（Intelligence Productization #3）

## 概要

「解決したい課題」を自由記述入力すると、GUILD AI の知能資産から最適な MD を提案する
**企業向け検索インターフェース**。

一般向けマーケットとは別の `/marketplace/pro` ページに配置し、
主動線（ナビ・FAB）には乗せず、サイドバーのフッタに控えめリンクを置く。

## フィルタ仕様

| フィルタ | 選択肢 | デフォルト |
|---------|--------|----------|
| カテゴリ | DataOps / LLMOps / RPA / 営業AI / すべて | すべて |
| 必要精度 | >90% / >95% / >99% / 指定なし | 指定なし |
| 月間コール数 | 〜10K / 〜100K / 100K+ / 指定なし | 指定なし |
| SLA | 99.5% / 99.9% / 99.99% / 指定なし | 指定なし |
| 価格帯 | Hobby / Pro Indie / Enterprise / 指定なし | 指定なし |

## ランキングロジック

1. 各アイテムに以下でスコアを付与：
   - クエリとタイトルの部分一致: +20pt
   - クエリと説明の部分一致: +10pt
   - S ランク: +15pt, A ランク: +8pt
   - (accuracyPct - 80) × 2: 精度ボーナス
   - avgLatencyMs > 200ms: ペナルティ
   - trustScore / 100: 信用ボーナス
2. フィルタ通過後にスコア降順でソート
3. 並び替え: 精度順 / レイテンシ順 / コスト順（フロント選択）
4. 上位 5 件を返す

## 結果カード

- 左: ランクカラープレート（S=金, A=赤, B=グレー）
- 中央: タイトル / 説明 / 精度バッジ / 平均レイテンシ / 推奨ティア / 価格
- 右: 2段 CTA
  - 「この知能で解決」→ `/asset/[id]`（赤丸）
  - 「ためしてみる」→ `/asset/[id]`（Hobby 枠）
  - 「法人で相談」→ `/business`

## 企業導線

- ページ末尾に「企業利用について相談する」ダークカード（金 CTA → `/business`）
- サイドバーフッタに「🏢 法人検索」テキストリンク（控えめ、主動線には乗せない）
