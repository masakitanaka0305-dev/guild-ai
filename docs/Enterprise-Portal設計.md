# Enterprise Portal 設計（#99 Autonomous Operator）

## 概要

法人・個人事業主が GUILD AI のプランを自分でセルフ申し込みできる
セルフサーブ・エンタープライズ・ポータル。
CC モックによるチェックアウトフローを含む。

## 実装ファイル

| ファイル | 役割 |
|----------|------|
| `src/app/business/page.tsx` | 既存の法人ページを拡張（チェックアウト CTA 追加） |
| `src/app/business/checkout/page.tsx` | クレジットカード申し込みフロー |

## プラン一覧

| プラン | 価格 | コール枠 | 想定ユーザー |
|--------|------|---------|-------------|
| Hobby | 無料 | 1,000/月 | 個人試作・学習 |
| Pro Indie | ¥1,980/月 | 50,000/月 | 個人事業主 |
| Enterprise | ¥0.03/call | 無制限 | 商用・SLA 保証 |

## チェックアウトフロー

```
/business → 「プランを選んで始める」→ /business/checkout
  → プランカード選択（radiogroup）
  → CC フォーム（番号 / 有効期限 / CVC / 名義）
  → バリデーション（Luhn チェック + 桁数）
  → 1.2秒モック処理
  → 完了画面 → /guild へ誘導
```

## CC フォームのバリデーション

- カード番号: Luhn アルゴリズム + 16桁
- 有効期限: MM/YY 形式 5桁
- CVC: 3桁数字
- 名義: 空白禁止

## セキュリティ注意

現在はモック実装。本番では:
- TLS 必須
- カード番号は DOM 経由で取得せず Stripe.js / PayPay 等の SDK を使用
- サーバー側でのみ決済 API を呼び出す
